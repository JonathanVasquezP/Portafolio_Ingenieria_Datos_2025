# 🔢 Calculadora Básica en Python

Este proyecto fue desarrollado en Google Colab como parte del curso **Fundamentos de la Ingeniería de Datos**.

## 🎯 Objetivo

- Simular una calculadora con operaciones básicas (suma, resta, multiplicación y división).
- Usar funciones en Python para cada operación.
- Ejecutar el flujo completo desde Colab.

## 📄 Archivo incluido

- `clase_m2_calculadora.ipynb`: cuaderno de Google Colab con el código.

## ▶️ Cómo usarlo

Puedes abrir el archivo en Google Colab directamente desde GitHub.

---

## 🧰 Herramientas usadas

- Python 3.x
- Google Colab
